> boilerplate (`react` `redux` `antd`)


*`npm` version >= `5.2.0`*


### Usage

`git clone https://github.com/MiRinZhang/react-boilerplate.git`


`cd react-boilerplate && yarn`

or

`cd react-boilerplate && npm i`


#### development

`yarn start`

or

`npm run start`


#### build

`yarn build`

or

`npm run build`
